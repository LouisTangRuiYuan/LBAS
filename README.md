# Registration to Registry

## 1. Install Python
Download and install Python on your laptop.

## 2. Change the region
Open `registration_to_registry.py` and change:

```python
REGION_NAME = "South West"
REGISTRY_SHEET_NAME = "South West"
```

Replace `South West` with your own region name.

## 3. Put the files in one folder
Place these files together in the same folder:
- `registration_to_registry.py`
- `run_windows.bat`
- `run_unix.sh`
- `Registry Database.xlsx`
- `Lan Berambeh Anak Sarawak 2026 UK Edition_ Registration Form (Responses).xlsx`

## 4. Run the script
### Windows
Double-click `run_windows.bat`

### macOS / Linux
Open Terminal in the folder and run:

```bash
./run_unix.sh
```
